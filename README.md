# product-category-classification

Product Category Classification

Ovaj projekat prikazuje ML model za automatsku klasifikaciju proizvoda po kategorijama na osnovu naziva proizvoda.

Cilj je da se ubrza i pojednostavi proces unosa proizvoda u online prodavnicu bez ručne klasifikacije.

Dataset

Korišćen je skup podataka sa nazivima proizvoda i pripadajućim kategorijama. Dataset se nalazi u folderu data.

Koristi test_model.py za testiranje ml modela, train_model.py za pokrtanja ml modela.
